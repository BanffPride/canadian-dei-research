# DEI Market Intelligence Research

Strategic analysis of the Canadian Diversity, Equity, and Inclusion consulting ecosystem.

## 📊 Project Overview

This repository contains a comprehensive market analysis of 80+ organizations in the Canadian DEI consulting space, including datasets, competitive analysis, and strategic insights.

## 📁 Contents

- **`/data`** — Raw datasets (Excel, CSV)
  - `competitor_dataset.xlsx` — Complete organization database
  - `pricing_benchmarks.csv` — Pricing analysis across service types
  - `service_saturation.csv` — Market saturation by offering

- **`/analysis`** — Strategic analysis documents
  - Market segmentation findings
  - Competitive landscape assessment
  - Strategic gap analysis

- **`/visualizations`** — Market maps and visual frameworks
  - Competitive positioning matrix
  - Market saturation heatmap
  - Service offering distribution

- **`/report`** — Full strategic reports
  - `Canadian_DEI_Strategic_Intelligence_Report.docx` — 16-section comprehensive analysis
  - Executive summary
